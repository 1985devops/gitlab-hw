   5. Для проверки домашнего задания преподавателем в личном кабинете прикрепите и отправьте ссылку на решение в виде md-файла в вашем Github.

### Задание 2

1. `Написал скрипт и настроил задачу на ежедневное резервное копирование домашней директории пользователя с помощью rsync и cron.`
2. `Резервная копия полностью зеркальная`
3. `Резервная копия размещается локально, в директории /tmp/backup`

```
#!/bin/bash

SOURCE="$HOME/"
DEST="/tmp/backup"

mkdir -p "$DEST"

if rsync -av --delete "$SOURCE" "$DEST" > /dev/null 2>&1; then
    logger "Backup Task: Success - mirror created at $DEST"
else
    logger "Backup Task: Error - backup failed"
fi


```
